timeBased {xts}R Documentation

Check if Class is Time-Based

Description

Used to verify that the object is one of the known time-based classes in R.

Usage

is.timeBased(x)
timeBased(x)

Arguments

x

object to test

Details

Current time-based objects supported are Date, POSIXct, chron, yearmon, yearqtr, and timeDate.

Value

Logical

Author(s)

Jeffrey A. Ryan

Examples

timeBased(Sys.time())
timeBased(Sys.Date())

timeBased(200701)
