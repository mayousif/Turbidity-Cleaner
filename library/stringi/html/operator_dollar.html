%s$% {stringi}R Documentation

C-Style Formatting with sprintf as a Binary Operator

Description

Provides access to base R's sprintf in form of a binary operator in a way similar to Python's % overloaded for strings.

Usage

e1 %s$% e2

e1 %stri$% e2

Arguments

e1

a single format string, see sprintf for syntax

e2

a list of arguments to be passed to sprintf or a single atomic vector

Details

Vectorized over e2.

e1 %s$% atomic_vector is equivalent to e1 %s$% list(atomic_vector).

Value

Returns a character vector.

Examples

"value='%d'" %s$% 3
"value='%d'" %s$% 1:3
"%s='%d'" %s$% list("value", 3)
"%s='%d'" %s$% list("value", 1:3)
"%s='%d'" %s$% list(c("a", "b", "c"), 1)
"%s='%d'" %s$% list(c("a", "b", "c"), 1:3)
